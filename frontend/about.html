About This Project

Student Management CRUD Application

This is a full-stack minor project utilizing the MEAN/MERN architectural patterns designed using Node.js, Express, MongoDB, and Bootstrap 5 frontend. Custom APIs manage application state dynamically through async/await JavaScript logic.

Tech Stack:
  • Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
  • Backend API: Node.js, Express.js
  • Database: MongoDB via Mongoose
  • Architecture: MVC (Model, View, Controller)

Developed as a 1st year BTech CSE minor project.
